# Hospital-Database-Analytics (Interactive Dashboard using Power BI)
## Project Objective
The goal of this project is to analyze hospital operational data and create an interactive Power BI dashboard. This helps hospital management understand patient trends, departmental performance, and financial KPIs, enabling better decision-making and improved efficiency.
